## Python Code Example


아래는 짝수와 홀수를 판별하는 간단한 파이썬 코드입니다.

```python   
def check_even_or_odd():
    # 사용자 입력
    number = int(input("Enter a number: "))
    
    # 짝수인지 홀수인지 확인
    if number % 2 == 0:
        print(f"{number} is an even number.")
    else:
        print(f"{number} is an odd number.")

# 함수 실행
check_even_or_odd()
```
